Detailed Description

template<jit::float_scalar T, bool SAFE_MATH = false>
class graph::pseudo_variable_node< T, SAFE_MATH >

Class representing a subexpression that acts like a variable.

Pseudo variable nodes treat sub trees as if they were a variable. This ensures that the expression returns zero when taking a derivative with something that is not itself.

Template Parameters
TBase type of the calculation.
SAFE_MATHUse Safe Math operations.
