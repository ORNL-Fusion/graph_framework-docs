Detailed Description

template<jit::float_scalar T, bool SAFE_MATH = false>
class graph::index_2D_node< T, SAFE_MATH >

Class representing a 2D index.

This class is used to implement 2D indexes into an array.

Indicies are selected by

x_norm' = (x - xmin)/dx (1) y_norm' = (y - ymin)/dy (2)

Template Parameters
TBase type of the calculation.
SAFE_MATHUse Safe Math operations.
